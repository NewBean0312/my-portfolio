PyeongChang2018 리뉴얼 웹사이트

평창2018 동계올림픽 사이트를 리뉴얼한 웹사이트입니다.
이 웹사이트는 모바일에서도 동작 가능한 반응형 디자인으로 제작되었으며,
예전에 평창 동계 올림픽을 재미있게 보았던 추억이 있어,
다른 사람들에게 다시 보여주고자 하는 마음으로 개발했습니다.

이 웹사이트에는 반응형 디자인을 공부하기 위해 제작하였습니다.

  • Function
    올림픽 소개 및 종목 소개, 뉴스와 contact 섹션, 구글 지도 api 사용, 모바일 버전 반응형 디자인
  • GitHub
    github.com/NewBean0312/PyeongChang2018-website-code
  • Skills
    HTML, CSS, 구글맵 API
